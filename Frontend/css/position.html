CSS Positioning Property

The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute, or sticky). 

        1. static   → Default, normal document flow
        2. relative → Positioned relative to itself
        3. fixed    → Positioned relative to viewport
        4. absolute → Positioned relative to nearest positioned ancestor
        5. sticky   → Switches between relative and fixed
